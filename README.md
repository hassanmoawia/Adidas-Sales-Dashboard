# Adidas-Sales-Dashboard

👟 Adidas Sales Performance Dashboard

📊 Project Overview

This Power BI project provides an end-to-end data analysis solution for Adidas sales performance. The dashboard integrates complex datasets across various dimensions—including regions, retailers, and product categories—to deliver actionable business insights.

🚀 Key Highlights

Executive KPI Tracking: Developed a centralized view for monitoring $899.90M in Total Sales, $332.13M in Operating Profit, and 2M Units Sold.

Retailer Performance Analysis: Isolated top-performing retailers, identifying West Gear and Foot Locker as the primary drivers of sales volume.

Regional & Channel Insights: Visualized sales distribution across five key U.S. regions, revealing the West region as the top contributor, and analyzed sales methods (In-store, Outlet, and Online).

Dynamic Trend Monitoring: Engineered "Total Sales by Day" time-series visualizations to track revenue fluctuations and identify high-growth periods.

🛠️ Technical Implementation

Data Integration: Merged and transformed multi-dimensional sales data to ensure a unified "Single Source of Truth" for reporting.

Advanced DAX: Authored custom DAX measures and calculated columns to drive high-performance KPI cards and profit margin analysis.

Interactive UI/UX: Implemented advanced interactive features including a Date Range slicer, Retailer/Region filters, and product-specific image selections to enhance stakeholder self-service capabilities.
